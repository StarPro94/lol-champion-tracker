# Refined Design System - Premium Game Collection

## Direction
- Premium game-inspired interface for a personal League of Legends tracker.
- Energetic, collectible, and expressive, but no longer chaotic or kitsch.
- The dancing dwarf stays absurd and memorable; the rest of the UI stays controlled.

## Visual language
- Layered dark backgrounds instead of flat pure black.
- Pink, cyan, yellow, and white remain the accent palette, used with restraint.
- Soft glows, textured overlays, glassy dark panels, and strong but selective contrast.
- Rounded cards, dense spacing, and clear hierarchy over floating stickers and novelty labels.
- Headings can stay italic and bold, but avoid fake luxury/slogan language.

## Product rules
- All visible UI copy must be in French, except proper champion names.
- Champion cards must show real Riot visuals first, with readable overlays and practical metadata.
- Local-first mode should be discreetly indicated, never explained in a large hero block.
- Decorative elements must always support the product mood or information density.

## Card system
- Champion cards use splash crops as the primary image.
- Text stack: champion name, French title, French role chips, French resource label, main action.
- Validated state should feel rewarding and clearly different, without overwhelming the image.
- Fallback to portrait icon if splash art fails.

## Layout cues
- 3 main zones: concise hero/progression, utility toolbar, champion grid.
- Background should feel inhabited and premium, not like an empty arcade room.
- Filters and summary blocks should look sharp, tactile, and immediately readable.
- Empty states should stay elegant and helpful, not theatrical.

## Tone
- Direct, playful, game-adjacent.
- No meaningless badges, no random stickers, no “luxury master list”, no filler hype copy.
